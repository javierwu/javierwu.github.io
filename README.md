## 🌐 如何使用

### 本地预览
```bash
# 方法1：直接打开
open index.html

# 方法2：使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 部署上线
1. **GitHub Pages**（免费）
   - 上传到GitHub仓库
   - 在Settings中启用Pages
   - 选择main分支

2. **Netlify**（免费）
   - 拖拽文件夹到netlify.com
   - 自动生成链接

3. **Vercel**（免费）
   - 连接GitHub仓库
   - 自动部署

## 🔧 自定义修改

### 修改截止日期
在 `index.html` 中找到：
```html
<p><strong>⏰ 截止时间：</strong>2026年X月X日 24:00</p>
```
将 `X月X日` 改为实际日期。

### 修改联系方式
在 `index.html` 的联系部分修改邮箱、电话等信息。

### 修改VI颜色
在 `styles.css` 的 `:root` 部分修改颜色变量。
